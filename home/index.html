Welcome to World of Wildlife

World Of Wildlife is the collection of wild animals which is supported by you all.

API

Description

World of Wildlife is a website made with GO language. All the collection of animals that are shown here are retrieved from the api. The api is also made with the GO language.


  • How the information is collected?

    The data is given by the users that have registered in this website which is added to the api. Any irrelevant data will be deleted.

  • Database

    All the data is stored in a sqlite database. It is easy to work with sqlite as it is fast, secure and light

  • Fast and Secure

    The website is hosted on Heroku and it is fast and the data of the user is secured.
